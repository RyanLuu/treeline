# Treeline

## Getting Started

Install dependencies

```
./install_deps
```

Build

```
make
```

Execute

```
bin/treeline
```

## Design

### Art

- cute and stylized (pixel? scribbly? watercolor?)
- minimal use of color

### Game

- imprecise and lenient rating system
- more than 4 keys
- possibly some notion of pitch accuracy
